
Para ejecutar esa app hace falta un servidor web, en mi caso tenia wampserver64 (apache, mysql , php)

configuración de la base dato:
	Nombre de basa de datos: tictactoe
	Tabla de base de datos: game_state

Antes de ejecutar la app hay que configurar el fichero bd.php que esta en la carpeta config :
entonces hay que conectar la base de datos : root, contraseña y nombre de base de datos 	

Para iniciar el juego: localhost/tic-tac-toe/index.php

Para jugar este juego de tres rayas he desarrollado dos opciones,  manual(dos personas jugando) y Maquina(Jugar contra el ordenador)

Jugar Manual: hay que introducir los nombres de los jugadores y luego inicia el juego 
Jugar con la Maquina: hay que introducir el nombre del jugador y luego inicia el juego  


El diseño de la app esta responsive si puede jugar en ordenador y movil
